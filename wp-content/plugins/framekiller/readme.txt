Plugin to prevent your site being open inside frames and iframes

== Description ==

With that plugin you can prevent your blog of being open inside frames and iframes. On plugin's options page you can allow same places of your blog to be open inside frames, for example, you can allow the home of your blog to be open inside frames but disallow others areas as: posts, pages, categories, tags, search results, etc.

== Installation ==

Follow the below steps to intall the plugin:
1- Send plugins's folder to plugins path, it should be /wp-content/plugins/
2- Access the blog panel and activate the plugin
3- Now, in the Settings area, a new link will appear "Framekiller", click on that link to go to plugin options page
4- Change the options you want and click on button Save Changes
